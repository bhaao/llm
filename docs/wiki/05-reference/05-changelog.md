# 变更日志

> **最后更新**: 2026-03-26  
> **项目版本**: v0.5.0

---

## [0.5.0] - 2026-03-26

### 新增
- 李群验证模块，实现信任根上移
- QaaS 质量验证服务
- 上下文分片支持（100K+ tokens）
- Bloom Filter 索引（O(1) 查找）
- zstd 压缩（93% 空间节省）
- 智能预取（N-gram 模式）

### 改进
- 服务层重构：拆分 coordinator.rs 为三个单一职责服务
- KV Cache 优化：Chunk-level 存储
- 多级缓存：L1 CPU + L2 Disk + L3 Remote
- 异步 I/O：全链路 async/await
- 文档结构优化：Wiki 风格文档集

### 修复
- P11 锐评指出的上帝对象问题
- 并发测试中的竞态条件
- 内存泄漏问题
- 死锁风险（规范锁顺序）

### 性能
- 李群聚合：100 节点 53µs（快 1880 倍）
- 距离计算：137ns（快 73000 倍）
- KV 并发写入：100 线程 ~50K ops/s
- 缓存命中率：85%+
