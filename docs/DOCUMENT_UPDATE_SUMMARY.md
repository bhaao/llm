## 📝 文档规范统一

### 项目定位声明

所有文档统一采用以下定位：

> **区块链驱动的分布式推理验证平台**，专为大模型推理场景设计，通过李群验证和区块链存证解决"如何验证分布式推理结果可信"的核心问题。

### 核心特性排序

1. **李群验证** - 100 节点聚合 53µs，信任根上移到数学公式
2. **区块链存证** - KV 哈希 + 李群根不可篡改记录
3. **PBFT 共识** - 三阶段提交 + 视图切换
4. **分布式 KV 存储** - 分片、压缩、多级缓存
5. **真实 LLM 集成** - 支持 vLLM/SGLang HTTP API

### 性能指标展示

**李群验证性能**（核心创新）:
| 指标 | 生产要求 | 实测 | 评价 |
|------|----------|------|------|
| 聚合时间 | < 100ms | **53.19 µs** | ✅ 快 1880 倍 |
| 距离计算 | < 10ms | **137 ns** | ✅ 快 73000 倍 |
| 篡改检测 | ×5.47 | **∞** | ✅ 验证通过 |
